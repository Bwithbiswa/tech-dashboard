🔗 Share & Access Guide

This dashboard is stored on OneDrive. Use OneDrive sharing to control who can view or edit.

🔐 You & Manager (Edit Access)

Right-click the file in OneDrive → Share → People with specific access → Set to Can edit for manager’s Intel email.

👁 Others (View Only)

Share with Can view permission — recipients open in browser, no edits possible to the file.

📺 Presenting

Click Present button (top right) for fullscreen mode. Or open in Edge → press F11 for browser fullscreen.

🔒 In-Browser Edit Mode (PIN Required)

Click Edit Mode in the header. Enter PIN to add/delete events. Changes save in this browser only.
To update the shared file permanently, edit the HTML file’s EVENTS array and save to OneDrive.

✅ No backend/server required. This is a self-contained HTML file. Works in any browser from OneDrive, email attachment, or USB drive.
